# The Nexus Terminal

一个赛博朋克风格的前端工具面板，集成：
1. 挥发记忆池（持久剪贴板）
2. 密码信道（图像 LSB 隐写）

界面语言为简体中文，视觉风格为黑底 + 霓虹青/紫的黑客终端。

## 功能概览

### 挥发记忆池（Volatile Memory Pool）
- 输入文本后按 Enter 保存，数据持久化到 `localStorage`
- 左键点击复制到系统剪贴板，右键删除并带“碎裂/淡出”动画
- 识别 URL 显示链接标记，识别代码片段显示不同边框色

### 密码信道（Cipher Channel）
- INJECT：上传图片 + 输入文本 → 通过 LSB 写入并导出 PNG
- EXTRACT：上传已注入图片 → 自动解析隐藏文本并以打字机效果展示
- 支持容量校验，文本过长会提示错误

## 项目结构

当前为纯前端结构，直接打开即可运行：
- `index.html` 页面结构 + 样式 + 逻辑（单文件版本）

## 使用方式

1. 直接双击打开 `index.html`（或用本地静态服务器打开）
2. 左侧写入文本保存到记忆池
3. 右侧选择 INJECT / EXTRACT 进行隐写或提取

## 注意事项

- 隐写导出格式固定为 PNG，以避免压缩损坏隐藏信息
- 建议使用分辨率较大的图片以获得更高容量
