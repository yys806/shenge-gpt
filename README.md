# 珅哥GPT 官方网站

这是一个展示珅哥GPT发展历史的官方网站，采用类似苹果官网的简约设计风格。

## 功能特点

- 响应式设计，适配各种设备
- 优雅的动画效果
- 智能对话功能
- 发展历史时间线
- 关于我们页面

## 技术栈

- HTML5
- CSS3
- JavaScript
- Vercel 部署

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/shenge-gpt.git
```

2. 打开 index.html 文件即可查看网站

## 部署

网站已通过 Vercel 部署，访问地址：https://shenge-gpt.vercel.app
